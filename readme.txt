        Enoncé :

> Classe Abstraite :
    La classe vetement est une classe Abstraite car elle comporte des méthodes abstraites.

> Heritage :
    Les classes rode, pantalon, t_shirt, jupe sont des classes filles dérivées de la classe vetement.

> Interface :
    On suppose que des types de vetements sont mixtes( peut etre porter par un garcon ou une fille ).
    La classe mixte est donc une interface.
